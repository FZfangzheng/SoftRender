# SoftRender

- ### 软光栅渲染器

  依赖库：

  - opencv-3.4.13（窗口展示）
  - eigen-3.4.0（数学运算）
  - [OBJ-Loader](https://github.com/Bly7/OBJ-Loader)（模型文件加载器）
  
  功能：
  
    - 绘制线、面、体
    - 透视投影相机
    - 纹理映射（透视矫正）
    - 点光源（Blinn-Phong光照模型）
    - shadow map 阴影
